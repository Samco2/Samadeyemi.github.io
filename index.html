GBS-ecommerce site

Django Python JavaScript HTLM CSS3 Git & Github SASS PostGres Javascript payment-API AWS Heroku

Tiaco is a fantastic, modern, technical and elegant  backend and frontend site. This site is basically designed as a beverage Ecommerce site with stripe and palpay payment integration. it uses session data to add selected item to cart for non-registered user while registered users could also save selected items for period of time as long as item still available for sale. At backend; admin  uploads new products, add price, discount or percentage discount, rating, generate promo code,, invoice generation and refund services are also available. The site monitor payment activities through API payment integration technology (called listen) and validate it with order details and users profile at backend using database called Postgress.
